① Crear un automata

  • Introducir Expresión

Escribir una expresión regular en el siguiente inputo o da clic en Generar Expresión Aleatoria para generar una expresión regular de forma aleatoria. Despues da clic en crear automata para poder el diagrama gráfico que es generado a partir de la expresión.

② Simular Automata

Escribir una cada en el input siguiente o dar clic en cadena aleatoria, Cadena valida o cadena invalida para que se genere una cadena aleatoria con el alfabeto que se esta definido en el punto número 1.

Ahora dar clic en Inicio para comenzar a leer un caracter de la cadena leer toda la cadena para leer toda los elementos faltantes de la cadena. Dar clic en atras para leer el elemento anterior restablecer para comenzar de nuevo y comenzar a leer toda la cadena.

El campo resalta el simbolo siguiente.
